MINECRAFT.INC: Quick intro

The game minecraft was first created by Swedish developer "Markus Persson", or known by Notch in the game and it's community. It was then released by Mojang in 2011. The game was purchased by microsoft in 2014 and became the best selling game of all time! it was simply the game with the most creativity and freedom of all time! selling more than 180 million copies across all platforms in 2019!
